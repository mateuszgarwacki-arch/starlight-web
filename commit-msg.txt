feat(labels): cutter mode (^MMC) + prefer https://localhost:9101 (S53)

- buildZpl now sends ^MMC so the printer feeds and cuts after each label
  (requires the cutter accessory fitted — the same one MS Access drives).
  Dropped ^MNY to avoid overriding the printer's existing gap calibration.

  for hosted apps; cert CN is "localhost"), then 127.0.0.1 / http variants.

User still needs Browser Print running + a one-time browser trust of
https://localhost:9101 (accept the localhost cert / allow the site).
