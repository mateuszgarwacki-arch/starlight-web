S34: Scope pack print moved from WO level to scope level

in WorkOrdersPanel. Scope-pack printing now lives on the scope header next
to the status badge, appearing once when woCount > 0.

Pack-mode printout restructured:
  1. Cover page (flow layout, spills to 2nd printed page if needed) —
     job number, scope item name, event date, scope description (full,
     verbatim), linked job items table (all items on scope, not filtered
     per-WO), pack contents summary
  2. Scope-level drawings (one per page, reusing ImagePage component).
     Fetched from tbl_wo_documents where scope_item_id = ? AND
     work_order_id IS NULL AND doc_type = 'drawing'
  3. Existing per-WO flow (divider -> brief -> cut lists -> drawings -> refs)
