Show freelancer's note in route-task modal header

Modal header was showing only the task title, not the description
the freelancer typed when logging the task — so the PM couldn't see
what the task was actually about without going back to the inbox.

Adds description to RoutableTask interface and renders it in a blue
left-bordered quote block under the metadata strip when present,
labelled "Freelancer's note". Scrollable, capped at ~128px high so
long notes don't blow out the header.
